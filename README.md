# Strix Launcher

A high-performance Minecraft client focused on speed, smooth gameplay, and optimization.

---

## ✨ Features

- ⚡ High FPS optimization
- 🎮 Smooth gameplay experience
- 🧩 Modular client system
- 🔧 Easy configuration
- 🌙 Clean modern UI
- 🚀 Fast startup and lightweight performance

---

## 📦 Installation

1. Download the latest release
2. Install Java 17 or newer
3. Launch `Strix Launcher`
4. Select your Minecraft version
5. Play

---

## 🛠 Requirements

- Java 17+
- Windows 10/11
- Minecraft Java Edition
